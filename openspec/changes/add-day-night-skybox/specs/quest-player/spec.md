## ADDED Requirements
### Requirement: Environment Configuration
The Quest Player SHALL support 'day' and 'night' environment modes, configurable via settings.

#### Scenario: Day Mode
- **WHEN** user selects "Day" environment
- **THEN** the scene renders a blue sky with sun and brighter lighting.

#### Scenario: Night Mode
- **WHEN** user selects "Night" environment
- **THEN** the scene renders a dark sky with stars and fog.
